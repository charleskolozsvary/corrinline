There is a great variety in how corrections can be specified with PDF annotations. However performance can be greatly enhanced with some standardization.

## Use dedicated annotations 
If the edit is to replace text, use the "Strikeout + Caret" annotation; if the edit is to remove, use the "Strikeout" annotation; and if the edit is
to insert directly next to a word, use the "Caret" annotation. Do not use "Highlight" as a catch-all.
